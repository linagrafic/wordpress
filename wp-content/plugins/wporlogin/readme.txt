== Description ==

WPOrLogin le permite personalizar fácilmente su página de inicio de sesión de administrador según sus necesidades.

Este plugin es compatible con todas las funcionalidades integradas de WordPress y los enlaces de cierre de sesión seguirán funcionando como deberían.

Si no se carga ningún logotipo, se muestra el logotipo predeterminado de WordPress.

**Características:**

* Logotipo personalizado
* Imagen de fondo
* Google reCAPTCHA v2
* Elimnar menú desplegable de Idioma en Página Login

== Installation ==

1. Unzip plugin files and upload them under your '/wp-content/plugins/' directory.
2. Resulted names will be: '/wp-content/plugins/wporlogin/'
3. Activate plugin at "Plugins" administration page.

== Frequently Asked Questions ==

= What is the recommened logo size? =

Less than or equal to 200px width and 84px height is the recommended logo size for your WordPress Login Page.
